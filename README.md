# SupplyTrack: Automated Hospital Bread Supply Management

SupplyTrack is a dedicated web application designed to streamline the invoicing and profit calculation process for consistent B2B supply operations. Built with the T3 Stack principles (Next.js, TypeScript) and powered by Supabase, this tool automates the manual, end-of-month calculation of profits, bakery costs, and invoice generation.
